Évolution de carrière

Hypothèses communes

  • Le point d’indice vaut \(4.92278\)
  • Le SMIC est à 1867 euros bruts
  • L’indemnité de résidence vaut
    • 3% à Palaiseau
    • 1% à Brest
  • Le salaire d’un mcf ne peut pas être inférieur à 2x SMIC
  • On considère la RIPEC C1 dans sa version 2025
Code 
import numpy as np
valeur_point_default = 4.92278 
SMIC = 1867

res_Brest = 1.01
res_palaiseau = 1.03
valeur_point = valeur_point_default

Calcul de l’ancienneté

On considère un recrutement après 2 années de postdoc

Dans la LPR,

  • les années de thèse comptent pour 3 ans (36 mois),
  • le docotat ajoute une année de bonification (12 mois)
  • et chaque année de postdoc compte (ici 24 mois).

L’ancienneté est donc de 72 mois.

Dans le MESR (source),

  • Échelon 1 dure 12 mois
  • Échelon 2 dure 34 mois
  • Échelon 3 dure 34 mois
  • Échelon 4 dure 34 mois
  • Échelon 5 dure 34 mois

Après deux ans de post doc on rentre donc avec 26 mois d’ancienneté dans l’échelon 3, dans lequel il reste 8 mois à faire.

Code 
years_mcf_default = np.array([0, 8, 12-8, 34 - (12-8), 34, 42, 34, 34])/12
Code 
RIPEC_2027 = 6400
RIPEC_2026 = (6400+4800)/2
RIPEC_2025 = 4800
  
indice_majore_mcf = np.array([589, 648,648, 698, 744, 774, 808, 835])

RIPEC_2025 = np.array([RIPEC_2025] * 8)/12  

RIPEC_ENSTA_2025 = 2000/12
RIPEC_ENSTA_2026 = 2800/12

Pour l’ENSTA

  • On part de l’IM 698
  • Deux options de RIPEC C1
    • C1 2025 (2000 euros)
    • C1 2026 (2800 euros)
  • 45 points d’augmentation tous les 3 ans
Code 
nombre_annees = 55
augmentation_tri = 45
Code 
number_years = int(np.floor(nombre_annees/3))
jump_indice = augmentation_tri
start_indice_default = 698
start_indice_ENS = 710

indice_majore_ENSTA_default = np.repeat(
    [start_indice_default + i * jump_indice for i in range(number_years)],
    3
)


years_ENSTA = np.array([0] + [1] * int(number_years-1))


Salaire_mcf_Pal = indice_majore_mcf * valeur_point * res_palaiseau
Salaire_mcf_Brest = indice_majore_mcf * valeur_point * res_Brest

### 2025
Salaire_mcf_Pal_2025 = Salaire_mcf_Pal + RIPEC_2025 
Salaire_mcf_Brest_2025 = Salaire_mcf_Brest + RIPEC_2025



### ENSTA
Salaire_ENSTA_Pal = indice_majore_ENSTA_default * valeur_point * res_palaiseau
Salaire_ENSTA_Brest = indice_majore_ENSTA_default * valeur_point * res_Brest

Salaire_ENSTA_Pal_prime = Salaire_ENSTA_Pal + RIPEC_ENSTA_2025
Salaire_ENSTA_Brest_prime = Salaire_ENSTA_Brest + RIPEC_ENSTA_2025

Salaire_ENSTA_Pal_prime_2026 = Salaire_ENSTA_Pal + RIPEC_ENSTA_2026
Salaire_ENSTA_Brest_prime_2026 = Salaire_ENSTA_Brest + RIPEC_ENSTA_2026

Plots

On considère un recrutement à 28 ans.

Palaiseau

Code 
import plotly.graph_objects as go
fig = go.Figure()

### IM 698

fig.add_trace(go.Scatter(x=28+np.cumsum(years_ENSTA), y=12*Salaire_ENSTA_Pal_prime, mode='lines+markers', name=f'ENSTA (IM 698)', line=dict(color='#d95f02', shape='hv'), marker=dict(symbol='circle')))



### 2025 figee
fig.add_trace(go.Scatter(x=28+np.cumsum(years_mcf_default), y=12*Salaire_mcf_Pal_2025, mode='lines+markers', name='MCF RIPEC 2025 figée', line=dict(color='#1f77b4', shape='hv'), marker=dict(symbol='circle')))




fig.add_hline(
    y=2*SMIC*12, 
    line_width=2, 
    line_dash="dash", 
    line_color="black",
    annotation_text="2x SMIC", 
    annotation_position="top right" 
)

############## Plotly
fig.update_layout(title='Évolution de la rémunération', xaxis_title='Âge', yaxis_title='Rémunération brute annuelle (€)', template='plotly_white', legend=dict(title='Légende'), font=dict(size=12), width=800, height=800)
fig.update_xaxes(showgrid=True, gridwidth=1, gridcolor='lightgrey')
fig.update_yaxes(showgrid=True, gridwidth=1, gridcolor='lightgrey')
fig.update_yaxes(range=[0, 80000])

# mo.ui.plotly(fig)
fig.show()
