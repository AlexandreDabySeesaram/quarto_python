Calcul de l’ancienneté

Hypothèses communes

  • Le point d’indice vaut \(4.92278\)
  • Le SMIC est à 1867 euros bruts
  • L’indemnité de résidence vaut
    • 3% à Palaiseau
    • 1% à Brest
  • Le salaire d’un mcf ne peut pas être inférieur à 2x SMIC
Code 
import numpy as np
valeur_point_default = 4.92278 
SMIC = 1867

res_Brest = 1.01
res_palaiseau = 1.03
valeur_point = valeur_point_default

On considère un recrutement après 2 années de postdoc

Dans la LPR,

  • les années de thèse comptent pour 3 ans (36 mois),
  • le docotat ajoute une année de bonification (12 mois)
  • et chaque année de postdoc compte (ici 24 mois).

L’ancienneté est donc de 72 mois.

Dans le MESR source,

  • Échelon 1 dure 12 mois
  • Échelon 2 dure 34 mois
  • Échelon 3 dure 34 mois
  • Échelon 4 dure 34 mois
  • Échelon 5 dure 34 mois

Après deux ans de post doc on rentre donc avec 26 mois d’ancienneté dans l’échelon 3, dans lequel il reste 8 mois à faire.

Code 
years_mcf_default = np.array([0, 8, 12-8, 34, 34, 42, 34, 34])/12

RIPEC

La RIPEC C1 doit atteindre 6400 euros en 2027 (source). Avec les retards de vote de budget, la RIPEC d’une année N peut être effective en juillet de cette année avec effet retroactif comme cela a eu lieu en 2025 (source). On peut donc choisir avance de compter la RIPEC 2026 pour l’année en cours ou choisir de ne compter pour 2026 que la RIPEC 2025.

On étudie donc trois scenarii

  • La RIPEC est figée à sa valeur de 2025 (contre la LPR)
  • La RIPEC reste à sa valeur 2025 en 2026
  • La RIPEC suit la LPR et on interpole entre les valeurs 2025 et 2027 pour 2026
Code 
RIPEC_2027 = 6400
RIPEC_2026 = (6400+4800)/2
RIPEC_2025 = 4800
  
indice_majore_mcf = np.array([589, 648,648, 698, 744, 774, 808, 835])

RIPEC_2026_2027 = np.array([RIPEC_2026]*2 + [RIPEC_2027] * 6)/12  
RIPEC_2025_2027 = np.array([RIPEC_2025]*2 + [RIPEC_2027] * 6)/12  
RIPEC_2025 = np.array([RIPEC_2025] * 8)/12  

RIPEC_ENSTA = 2800/12

Pour l’ENSTA

  • On part de l’IM 678
  • RIPEC C1 = 2800 euros (2026)
  • 40 points d’augmentation tous les 3 ans
Code 
nombre_annees = 55
augmentation_tri = 40
Code 
number_years = int(np.floor(nombre_annees/3))
jump_indice = augmentation_tri
start_indice_default = 678

indice_majore_ENSTA_default = np.repeat(
    [start_indice_default + i * jump_indice for i in range(number_years)],
    3
)


years_ENSTA = np.array([0] + [1] * int(number_years-1))


Salaire_mcf_Pal = indice_majore_mcf * valeur_point * res_palaiseau
Salaire_mcf_Brest = indice_majore_mcf * valeur_point * res_Brest

### 2025
Salaire_mcf_Pal_2025 = Salaire_mcf_Pal + RIPEC_2025 
Salaire_mcf_Brest_2025 = Salaire_mcf_Brest + RIPEC_2025

### 2025 2027
Salaire_mcf_Pal_2025_2027 = Salaire_mcf_Pal + RIPEC_2025_2027
Salaire_mcf_Brest_2025_2027 = Salaire_mcf_Brest + RIPEC_2025_2027

### 2026 2027
Salaire_mcf_Pal_2026_2027 = Salaire_mcf_Pal + RIPEC_2026_2027
Salaire_mcf_Brest_2026_2027 = Salaire_mcf_Brest + RIPEC_2026_2027

### ENSTA
Salaire_ENSTA_Pal = indice_majore_ENSTA_default * valeur_point * res_palaiseau
Salaire_ENSTA_Brest = indice_majore_ENSTA_default * valeur_point * res_Brest

Salaire_ENSTA_Pal_prime = Salaire_ENSTA_Pal + RIPEC_ENSTA
Salaire_ENSTA_Brest_prime = Salaire_ENSTA_Brest + RIPEC_ENSTA
